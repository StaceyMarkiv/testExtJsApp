Описание работы приложения

Данный проект показывает работу с фреймворком ExtJS 5.1.2.

База данных PostgreSQL, работа с базой данных через PHP 7.2 (дамп базы данных лежит в папке resources\sql, параметры подключения к базе в файле resources\ini\dbconnect.ini).

Приложение состоит из следующих компонентов:

  • панель авторизации
  • основная панель приложения
  • панель администратора для управления учетными записями
  • дополнительная панель для редактирования вспомогательной информации / отображения графиков

Панель авторизации

Данная панель появляется при исходной загрузке приложения, все остальные панели скрыты. После ввода логина и пароля панель авторизации скрывается, появляются панели, соответствующие роли авторизованной учетной записи (админ, редактор, пользователь).

Если логин и/или пароль были указаны неверно, будет выведено соответствующее сообщение, форма авторизации будет очищена.

Панель администратора

Появляется только для учетной записи с ролью администратора.

Панель представляет собой простую таблицу (grid), содержащую перечень зарегистрированных учетных записей.

Все ячейки каждой записи редактируемые: столбец "Роль" - выпадающий список, остальные - простые текстовые поля. Пароль вводится как комбинация обычных букв/цифр/символов, а при сохранении в базу данных хэшируется.

Удалить любую учетную запись можно по кнопке удаления слева от записи.

Учетные записи могут иметь одну из следующих ролей:

admin Полный доступ ко всем функциям, редактирование учетных записей
editor Может редактировать данные основной формы и наполнение выпадающих списков
user Может редактировать только данные основной формы

Основная панель приложения

Панель представляет собой простую таблицу (grid), содержащую основные данные приложения. На этой панели можно производить различные манипуляции с данными, для обработки и хранения которых создавалось приложение.

В заголовки столбцов таблицы встроен стандартный фильтр при помощи плагина "gridfilters". На нижнюю панель добавлена кнопка с меню "Фильтры", выполняющая аналогичную фильтрацию. Логика работы кнопки прописана мной самостоятельно. Данная кнопка была создана, так как работа со встроенными с заголовки столбцов фильтрами показалась не очень удобной.

Меню кнопки "Фильтры" включает в себя следующие пункты:

  • Пользователи
  • Образование
  • Город
  • Возраст
  • Очистить фильтры

Фильтры "Пользователи", "Образование", "Город" представляют собой списки, в которых галочками можно отметить необходимые для отображения пункты. У каждого из данных подменю внизу панели есть кнопка "Выбрать все / Убрать все", расширяющая стандартную SegmentedButton. Данная кнопка позволяет проставить/убрать сразу все отметки в фильтре.

Фильтр "Возраст" представляет собой фильтр по возрасту на основе radiogroup. Он позволяет отобрать записи, относящиеся к одной из перечисленных возрастных групп. Можно выбрать только одну возрастную группу за раз.

Все фильтры можно использовать одновременно. Каждый последующий фильтр накладывается на уже отфильтрованные предыдущим фильтром записи.

Пункт "Очистить фильтры" позволяет очистить все наложенные на хранилище фильтры сразу, включая текстовый фильтр по содержимому.

Текстовый фильтр по содержимому находится на нижней панели рядом с кнопкой "Фильтры". В поле ввода можно вводить любые символы, цифры, буквы русского/английского алфавита. В результате работы фильтра останутся записи, которые хотя бы в одном из полей содержат введенный текст. Рядом с полем ввода находится кнопка очистки данного фильтра.

Также на нижней панели имеются еще две управляющие кнопки: "Добавить" и "Действия".

Кнопка "Добавить" позволяет добавить новую запись в таблицу. При нажатии на нее открывается всплывающее окно с полями для ввода необходимой информации. По кнопке "Добавить" на нижней панели всплывающего окна введенная информация отправляется в базу данных.

Кнопка "Действия" позволяет управлять цветовым оформлением записей основной таблицы. Ее меню содержит следующие пункты:

  • Показать пользователей с машиной
  • Показать машины с данными по умолчанию
  • Показать пользователей с незаполненными данными

Каждый из данных пунктов выделяет цветом соответствующие записи в таблице.

В заголовке панели находятся две кнопки: [ ? ] и "Выход". По нажатию на кнопку [ ? ] открывается всплывающее окно с описанием особенностей работы приложения. Кнопка "Выход" предназначена для смены учетной записи. По нажатию на нее все видимые панели скрываются, появляетсяя панель авторизации.

Ячейки всех столбцов основной панели (за исключением столбца "Возраст") редактируемые. В столбце "Образование" редактор - выпадающий список, для столбцов "Пользователь" и "Город" созданы собственные pickers. Столбец "Машина" - столбец с галочками.

В столбце "Возраст" указывается текущий возраст пользователя, расчитанный на основании его даты рождения. Конвертация даты рождения в возраст производится в модели (файл mainPageModel.js). Данные в столбце не редактируются напрямую, а пересчитываются в зависимости от указанного возраста пользователя (задается во всплывающем окне в информацией о пользователе).

Если у пользователя сегодня день рождения, запись его возраста подсвечивается зеленым, а при наведении на ячейку с возрастом появляется всплывающая подсказка с соответствующим уведомлением.

Самый левый столбец - actioncolumn, столбец с управляющими кнопками к каждой записи. Кнопка "Удалить" позволяет удалить выбранную запись. Кнопка "Информация" открывает всплывающее окно с полной информацией по каждому пользователю. Форма во всплывающем окне редактируемая.

Дополнительная панель

Для учетных записей с ролями admin и editor доступна возможность редактирования наполнения выпадающих списков столбцов "Образование" и "Город". По щелчку правой кнопкой мыши на ячейке столбцов "Образование" и "Город" появляется контекстное меню со следующими пунктами:

  • Редактировать ступени образования (только для столбца "Образование")
  • Редактировать список городов (только для столбца "Город")

По щелчку на любой из данных пунктов контекстного меню открывается боковая панель с соответствующей формой для редактирования. Форма позволяет добавить город / ступень образования, отредактировать или удалить любую из имеющихся записей.
